# Quickstart

Get IntentusNet running in 5 minutes with this quickstart guide.

## Prerequisites

- Python 3.9+
- IntentusNet installed (`pip install intentusnet`)

## Step 1: Create a Simple Agent

Create `my_agent.py`:

```python
from intentusnet import (
    IntentusRuntime,
    BaseAgent,
    IntentEnvelope,
    AgentResponse,
    AgentDefinition,
    Capability,
    IntentRef,
)

# Define an agent that processes text
class TextProcessorAgent(BaseAgent):
    def __init__(self, router):
        # Define capabilities inside the agent constructor
        definition = AgentDefinition(
            name="text-processor",
            version="1.0",
            capabilities=[
                Capability(
                    intent=IntentRef(name="ProcessTextIntent", version="1.0"),
                    inputSchema={"text": "string"},
                    outputSchema={"length": "number", "words": "number"}
                )
            ],
        )
        super().__init__(definition=definition, router=router)

    def handle_intent(self, env: IntentEnvelope) -> AgentResponse:
        text = env.payload.get("text", "")
        processed = {
            "original": text,
            "length": len(text),
            "words": len(text.split()),
            "uppercase": text.upper()
        }
        return AgentResponse.success(
            processed,
            agent=self.definition.name,
            trace_id=env.metadata.traceId,
        )
```

## Step 2: Run the Intent

Add to `my_agent.py`:

```python
def main():
    # Create runtime with recording enabled
    runtime = IntentusRuntime(enable_recording=True)

    # Register the agent using a factory function
    # The runtime injects the router automatically
    runtime.register_agent(TextProcessorAgent)

    # Create a client to send intents
    client = runtime.client()

    # Send the intent - the client builds the envelope for you
    print("Executing intent...")
    response = client.send_intent(
        "ProcessTextIntent",
        {"text": "Hello, IntentusNet! This is a test message."},
        tags=["quickstart", "demo"],
    )

    # Print results
    print(f"\nStatus: {response.status}")
    print(f"Agent: {response.metadata.get('agent', 'N/A')}")
    print(f"Trace ID: {response.metadata.get('traceId', 'N/A')}")
    print(f"\nPayload:")
    import json
    print(json.dumps(response.payload, indent=2))

if __name__ == "__main__":
    main()
```

## Step 3: Run It

```bash
python my_agent.py
```

Expected output:

## Step 4: Inspect the Execution

Check the recorded execution:

```bash
ls -la .intentusnet/records/
```

View an execution record:

```bash
cat .intentusnet/records/*.json | python -m json.tool | head -50
```

## Step 5: Replay the Execution

Replay returns the recorded output without re-execution:

```python
from intentusnet import FileExecutionStore, ReplayEngine

# Load the execution record
store = FileExecutionStore(".intentusnet/records")
records = list(store.list_all())

if records:
    record = store.load(records[0])  # Load first record

    # Create replay engine
    engine = ReplayEngine(record)

    # Check if replayable
    is_ok, reason = engine.is_replayable()
    print(f"Replayable: {is_ok}")

    if is_ok:
        result = engine.replay()
        print(f"Replayed payload: {result.payload}")
else:
    print("No records found")
```

## Complete Working Example

Here's the complete `my_agent.py` file:

```python
from intentusnet import (
    IntentusRuntime,
    BaseAgent,
    IntentEnvelope,
    AgentResponse,
    AgentDefinition,
    Capability,
    IntentRef,
)
import json

class TextProcessorAgent(BaseAgent):
    def __init__(self, router):
        definition = AgentDefinition(
            name="text-processor",
            version="1.0",
            capabilities=[
                Capability(
                    intent=IntentRef(name="ProcessTextIntent", version="1.0"),
                    inputSchema={"text": "string"},
                    outputSchema={"length": "number", "words": "number"}
                )
            ],
        )
        super().__init__(definition=definition, router=router)

    def handle_intent(self, env: IntentEnvelope) -> AgentResponse:
        text = env.payload.get("text", "")
        processed = {
            "original": text,
            "length": len(text),
            "words": len(text.split()),
            "uppercase": text.upper()
        }
        return AgentResponse.success(
            processed,
            agent=self.definition.name,
            trace_id=env.metadata.traceId,
        )

def main():
    # Create runtime with recording enabled
    runtime = IntentusRuntime(enable_recording=True)

    # Register the agent
    runtime.register_agent(TextProcessorAgent)

    # Create a client and send the intent
    client = runtime.client()
    response = client.send_intent(
        "ProcessTextIntent",
        {"text": "Hello, IntentusNet! This is a test message."},
        tags=["quickstart", "demo"],
    )

    # Print results
    print(f"Status: {response.status}")
    print(f"Agent: {response.metadata.get('agent', 'N/A')}")
    print(f"\nPayload:\n{json.dumps(response.payload, indent=2)}")

if __name__ == "__main__":
    main()
```

## What Just Happened?

1. **Agent Registration**: You created and registered an agent with a capability using `runtime.register_agent()`
2. **Client Creation**: The runtime provides a client via `runtime.client()`
3. **Intent Sending**: The client's `send_intent()` method builds the envelope and routes it
4. **Deterministic Routing**: The router matched your intent to the agent based on capabilities
5. **Execution Recording**: Every step was recorded to `.intentusnet/records/`

## Execution Flow

```
client.send_intent("ProcessTextIntent", {...})
        │
        ▼
┌───────────────────┐
│   IntentusClient  │  Builds IntentEnvelope
└─────────┬─────────┘
          │
          ▼
┌───────────────────┐
│    Transport      │  Routes to IntentRouter
└─────────┬─────────┘
          │
          ▼
┌───────────────────┐
│   IntentRouter    │  Finds agent by capability
└─────────┬─────────┘
          │
          ▼
┌───────────────────┐
│ TextProcessorAgent│  Executes handle_intent()
└─────────┬─────────┘
          │
          ▼
    AgentResponse
```

## Key Concepts Demonstrated

| Concept | What You Saw |
|---------|--------------|
| `IntentusRuntime` | Central runtime that wires everything together |
| `runtime.register_agent()` | Registers an agent with the runtime |
| `runtime.client()` | Returns a client for sending intents |
| `client.send_intent()` | Sends an intent and returns a response |
| `AgentResponse` | Standardized response format |
| Recording | Automatic capture to `.intentusnet/records/` |
